# Interesting Architectures

### OrbitDB

You spin up an IPFS instance for persistence

> [OrbitDB](https://github.com/orbitdb/orbitdb) is a **serverless, distributed, peer-to-peer database**. OrbitDB uses ++[IPFS](https://ipfs.tech/)++ as its data storage and ++[Libp2p Pubsub](https://docs.libp2p.io/concepts/pubsub/overview/)++ to automatically sync databases with peers.
